# 知识库项目记忆

## 项目背景
这是一个基于 Claude Code 的可复用知识库系统，核心理念是将知识库当代码管理，让 LLM 从"检索者"变为"知识编译器"。

主要用途：
- 个人知识管理和积累
- 技术/学术资料的结构化存储
- 知识的复用和演进

## 用户偏好
- 使用简体中文进行所有交流
- 知识条目保持原子性：一个文件只讲一个概念
- 回答简洁、结构化，便于后续吸收
