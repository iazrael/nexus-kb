# Idea File

## 定义
在 LLM agent 时代的一种新型知识传播方式——分享想法而非代码，让 AI agent 根据具体需求从零定制和构建系统。

## 核心要点
1. **超越代码仓库**：与其开源一个具体的代码仓库，不如写一个"idea file"（如 GitHub Gist），其中包含核心思路和设计模式
2. **定制化构建**：将 idea file 丢给 agent，agent 会根据具体需求定制和构建整个系统，每个人得到的实现都不一样，但核心模式一致
3. **知识传播范式转移**：从"给你一个成品让你 fork"变成"给你一个想法让你的 AI 帮你从零构建"，更具适应性和灵活性

## 边界条件
- **依赖 agent 能力**：需要 agent 具备足够的理解和执行能力来从想法构建系统
- **质量差异风险**：不同 agent 或配置可能产生质量差异很大的实现
- **缺乏标准化**：缺乏统一的标准和最佳实践，可能导致混乱
